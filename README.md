# 抖音主播内容整理（douyin_output）

对一批抖音财经 / 认知类主播的视频内容进行采集、整理与归档，把每条视频的口语内容沉淀为可检索、可回溯的 Markdown 文本，方便后续检索、复盘与学习。

## 目录结构

```
.
├── 0主播主页信息/          # 各主播主页链接汇总
├── _logs/                  # 采集流水线运行日志
└── <主播名>/               # 每位主播一个目录（见下方列表）
    ├── index.md            # 视频索引表：序号 / 链接 / 标题 / 写入时间
    ├── summary.json        # 每条视频的处理记录（url / title / file / writtenAt / status）
    ├── video_urls.json     # 视频原始链接列表
    └── *.md                # 单条视频的文字整理（文件名 = 视频标题 + 标签 + 主播名）
```

## 文件说明

- **`index.md`**：以表格形式列出该主播已整理的全部视频，含可点击的站内跳转链接，是最方便的浏览入口。
- **`summary.json`**：结构化的处理结果记录，包含每条视频的状态（`ok` 表示成功），便于程序化检查与增量更新。
- **`video_urls.json`**：采集到的视频原始 URL 与标题。
- **内容 `.md`**：每条视频对应的文字稿。文件名保留了原标题与话题标签（如 `财经 股票 任泽平`），便于通过文件名直接检索主题。

## 数据更新

本仓库通过 Git 跟踪增量更新，采集脚本持续运行后会生成新的内容文件，定期提交即可：

```bash
git add -A
git commit -m "Update: <说明>"
git push
```

- 本仓库内容均来源于抖音公开视频，版权归原作者 / 主播所有，整理文本仅供个人学习与检索，不作任何商业用途。
