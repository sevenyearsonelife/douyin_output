智能体时代，如何衡量 AI 公司的价值？  
智能体时代来了，但很多人还看不懂 AI 公司。  
问题在于衡量公司价值的工具没用对，思维还停留在互联网时代，旧尺子量不了新世界。  
AI 不是风口，是海啸，换尺子才能量出新世界的价值。

以前互联网时代是怎么衡量公司价值的？  
当时投资人的尺子叫 DAU 和 ARPU，日均活跃用户和单个用户收入。  
日活用户数乘以单个用户收入，就等于互联网公司的价值。  
日活越多，每个用户创造的收入越高，公司就越值钱。  
这把尺子量出了微信、抖音、美团的时代价值。

现在 AI 已经进入第二阶段，Agent 时代。  
这把尺子量不准了，原因不在技术，而是 AI 时代的生意本质变了。  
移动互联网做的是注意力生意，谁抢到用户时间，谁就赚钱。  
但 AI 不一样，AI 的核心不是让你多看一会，而是帮你把事干了。  
一个 AI 帮你审了100份合同，另一个 AI 帮你刷了100条短视频，消耗的时间一样，DAU 一样，但价值完全不同。  
DAU 分不出这种差异。

有人会说，那任老师，AI 消耗 token 的，那用 token 来衡量是不是就准确的？  
我告诉你，这种观点是错的。  
token 只能量消耗，量不了产出。  
1000个 token 问天气如何，和1000个 token 写一份行业报告，消耗相同，但产生的价值天壤之别。

今年龙虾的爆火就是这个信号，它不是模型，它是智能体，不等你输入，自己干活。  
这说明一件事，AI 的战场从参数竞争转向了执行力竞争，用户从工具使用者变成了任务发包方，AI 从聊天工具变成了数字员工，旧框架已经跟不上变化了。

我们再来看看新尺子长什么样。  
百度创始人李彦宏在 Create 2026 大会上提出一个新指标，DAA，Daily Active Agents，日活智能体数。  
这把新尺子有三个刻度：  
第一个刻度，DAA 规模，每天有多少智能体在跑，这不是统计人，是统计数字员工。  
一个平台上跑着10万个 DAA，说明有10万个智能体在每天上班。  
规模越大，生态越繁荣。

第二个刻度，任务完成率。  
智能体跑起来不等于有价值，关键看任务完成了多少，结果交付了多少。  
这是有效劳动的量化，只启动不交付的智能体是成本，不是价值。

第三个刻度，单任务价值。  
每个任务创造了多少实际价值？  
一个智能体帮企业完成采购谈判和帮用户查天气，任务价值相差悬殊。  
这个指标回答了 AI 产出能不能变现。

三个刻度放在一起，就是 Agent 时代的估值公式：  
DAA 乘以任务完成率乘以单任务价值，等于 AI 公司的真实产出。  
这不是猜一家 AI 公司值多少钱，而是量它每天创造了多少有效价值。

从模糊猜估值到精确量产出，换了尺子，能看见什么？  
尺子一换，视野就变了。  
李彦宏在 Create 2026 上做了一个预测，未来全球 DAA 可能是 Meta DAU 的数倍，超过100亿。  
Meta 的 DAU 约32亿。  
未来，一个人可以同时运行多个智能体，一家企业可以部署成百上千个智能体。  
智能体的数量天花板远超人口天花板。  
如果这个预测成真，Agent 时代的经济规模将远超移动互联网时代。

这对投资行业意味着两件事：  
第一，旧的评估标准全部失效。  
还在看用户数、算力消耗量、融资金额来评估 AI 公司，就像用磅秤量黄金，工具不对，结果毫无意义。

第二，必须建立新框架。  
真正值得问的是，这个平台每天跑着多少智能体？任务完成率多高？每个任务创造了多少可量化的价值？  
能回答这三个问题的公司，才是 Agent 时代的真资产。

移动互联网时代，抢的是注意力。  
Agent 时代，抢的是任务执行权。  
谁的智能体跑得最多、干得最好、创造价值最高，谁就是赢家。
