我相信你一定见过这种一个词根记多个单词的爆款视频吧。  
比如说这里，port 是运输，import 是进口，export 是出口。  
但是 port 是搬运，export 是驱逐出境。  
其实这是一个巨大的流量骗局。

我做过11年的单词记忆培训，见过无数用一个词根记很多个单词的这种案例去吸引别人买课的人。  
实际上词根能覆盖的单词很有限，约占单词总量里面的30%，70%的单词没有词根。  
这是2024年的报告。

你肯定很纳闷，为什么没有人愿意告诉你真相呢？  
词根记忆法适用的单词非常有限，像高中的大量单词基本上都没有词根。  
要是你只会词根的话，就麻烦大了。

由于词根覆盖不到很多单词，导致很多词根书刻意地乱造词根，为了牵强附会来让那些书看上去更合理，尽可能覆盖到更多的单词，这就造成了一种巨大的误导。  
所以词根的定位应该是适用于30%的包含词根的单词。

就是很多人为什么拿着一本词根背单词的时候，一个词根带五六个单词，轻松就记住了。  
然后一回到正常书，发现70%的单词，哎，怎么好像都没见过呢？  
是因为词根书里面覆盖不到。

真正的解决这个问题，我用了三年的时间打造了这本书。  
这本书所有有词根的单词，我都会帮你用词根记。  
没有词根的，我会想各种办法帮你记住。

你说这个单词 consist，c o n 是共同词根，s i 可以看成四方形，s t 是石头。  
想让大家共同把四方形的石头堆积成了金字塔，那么你很快就记住了这个单词，consist，组成、包含。  
金字塔有很多四方形的石头构成，这个单词的意思组成构成。

单词 consult 可以看成 c o n 共同，s u 数，l t 聊天，想要有共同诉求的人在一起聊天，找专业人士咨询，记住它的意思是咨询。  
那么一个单词，一个漫画，记得特别牢。

词根，这个书里面也会教你们怎么记。  
比如说 c o n 葱，洋葱加个 g 就是王思聪，国民老公，共同老公就可以记住了。

其他没有词根或者词根很少见的，在这本书里面都有非常好记的解，比如 ecology。  
一考了记一下生态学，一考了发现没记住，生态就记住了。

elder 长者，态度恶劣的批评年轻人，耳朵听到很难受，这个单词长者 elder。  
emperor 啊，是这个皇帝皇帝按老婆的做法去做，怕惹对方生气。

你看这些单词，我们其实都有啊，更强大的一种方法叫 link 记忆法，就是不管你有没有词根，或者你词根记不住，我们都有办法解决。
